In this module, you will progressively build your first enclave application and gain experience with some of the basic concepts involved in the Nitro Enclaves workflow, including: developing for Nitro Enclaves, building an enclave image file (EIF), launching an enclave, using the secure local vsock for communication between the parent instance and your enclave, working with debug mode, cryptographic attestation, and troubleshooting errors.
